# 智能字幕获取策略说明

## 📋 概述

本系统实现了智能的两步字幕获取策略，自动选择最优方案：

1. **优先策略**：使用YouTube原始字幕（人工或自动CC）
2. **备选策略**：Whisper ASR转录

## 🎯 工作流程

```
开始处理视频
    ↓
尝试下载YouTube字幕
    ↓
    ├─── 有字幕？
    │    ├─ 是 → ✓ 使用原始字幕（快速）
    │    │        └─ 跳过Whisper转录
    │    │        └─ 进入翻译阶段
    │    │
    │    └─ 否 → ⚠ 启用Whisper转录
    │             └─ GPU转录（耗时）
    │             └─ 进入翻译阶段
    ↓
继续处理...
```

## 🔍 详细说明

### 场景1：视频有原始字幕 ✓

**检测过程**：
```
2026-01-29 10:33:55 - downloader - INFO - ✓ en 人工字幕下载成功
2026-01-29 10:33:56 - main - INFO - ✓ 发现YouTube原始英文字幕（人工）
2026-01-29 10:33:56 - main - INFO -   跳过Whisper转录，节省时间和GPU资源
2026-01-29 10:33:56 - main - INFO -   ✓ 转换成功: cache/VIDEO_ID/subtitle.en.srt
```

**优势**：
- ⚡ **速度快**：5秒内完成
- 💰 **省资源**：不占用GPU
- ✓ **质量好**：人工字幕质量通常很高

**字幕类型识别**：
- **人工字幕**：由UP主或YouTube社区贡献
- **自动字幕(CC)**：YouTube自动生成的字幕
- 系统会在日志中明确标注字幕类型

### 场景2：视频无原始字幕 ⚠

**检测过程**：
```
2026-01-29 10:34:00 - downloader - WARNING - ✗ 该视频不提供 en 字幕
2026-01-29 10:34:00 - main - INFO - ✗ 未找到YouTube原始字幕
2026-01-29 10:34:00 - main - INFO - ════════════════════════════════════════
2026-01-29 10:34:00 - main - INFO - 使用Whisper ASR生成英文字幕
2026-01-29 10:34:00 - main - INFO -   模型: large-v3
2026-01-29 10:34:00 - main - INFO -   设备: cuda
2026-01-29 10:34:00 - main - INFO -   这可能需要几分钟时间，请耐心等待...
2026-01-29 10:34:00 - main - INFO - ════════════════════════════════════════
[Whisper转录进度...]
2026-01-29 10:38:00 - main - INFO - ✓ Whisper转录完成
```

**适用场景**：
- 新上传的视频（还没生成CC）
- 小众语言视频
- UP主禁用了自动字幕
- 需要最高准确度的专业内容

**Whisper优势**：
- 🎯 **准确度高**：large-v3模型精度最高
- 🌐 **多语言支持**：识别英语口音变体
- 🔧 **可定制**：可调整模型大小和参数

## 🛠️ 配置选项

### 强制使用Whisper（不推荐）

如果你想强制使用Whisper而不使用YouTube字幕，可以修改配置：

```yaml
# config.yaml
downloader:
  download_original_subtitles: false  # 改为false
```

### 调整Whisper参数

```yaml
transcriber:
  model: "large-v3"    # tiny/base/small/medium/large/large-v2/large-v3
  device: "cuda"       # cpu/cuda
  language: "en"       # 源语言
  fp16: true          # GPU加速
```

## 💡 最佳实践

### 建议配置（默认）

```yaml
downloader:
```

## 🔧 故障排查

### Q: 为什么跳过了Whisper？

A: 说明视频有原始字幕，这是正常且期望的行为。查看日志：
```
✓ 发现YouTube原始英文字幕（人工）
  跳过Whisper转录，节省时间和GPU资源
```

### Q: 如何知道使用了哪种字幕？

A: 查看日志输出：
- `✓ en 人工字幕下载成功` - 使用人工字幕
- `✓ en 自动字幕(CC)下载成功` - 使用YouTube CC
- `使用Whisper ASR生成英文字幕` - 使用Whisper转录

### Q: 原始字幕质量不好怎么办？

A: 可以手动删除下载的字幕，强制使用Whisper：
```bash
# 删除缓存的字幕
rm cache/VIDEO_ID/*.vtt
rm cache/VIDEO_ID/subtitle.en.srt

# 重新处理
python main.py --url "YOUR_URL"
```
